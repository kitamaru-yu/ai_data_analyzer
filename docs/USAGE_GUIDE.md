# 企業データ分析システム - 使用ガイド

## 🚀 クイックスタート

### 1. セットアップ
```bash
# 必要なパッケージをインストール
pip install -r requirements.txt

# .envファイルにAPI keyを設定
# .env ファイルを開いて、your_openai_api_key_here を実際のAPIキーに置き換えてください
```

### 2. システム起動
```bash
# 方法1: バッチファイルで起動（推奨）
run_system.bat

# 方法2: 直接起動
python main.py                 # コマンドライン版
streamlit run streamlit_app.py  # Webアプリ版
```

## 📊 主要機能

### データ分析機能
- ✅ CSV/テキストファイルの読み込み
- ✅ データ構造の自動分析
- ✅ AI による詳細分析（GPT-4使用）
- ✅ 統計的分析（相関、分布、異常値検出）

### 可視化機能
- ✅ 相関行列ヒートマップ
- ✅ 分布図・ヒストグラム
- ✅ 散布図・箱ひげ図
- ✅ カテゴリ分析・棒グラフ
- ✅ トレンド分析・線グラフ
- ✅ AI推奨の統合可視化

### ビジネス戦略提案
- ✅ 現状課題・機会の特定
- ✅ 優先度付き行動計画
- ✅ メリット・デメリット・リスク分析
- ✅ 数値目標・KPI設定
- ✅ 短期・中期・長期ロードマップ

## 🚨 注意事項

### API使用料金
- GPT-4使用時は特に料金に注意
- 大量データ処理時は事前に見積もり
- APIキーの使用量制限を確認

### データセキュリティ
- 機密データの取り扱いに注意
- 必要に応じてローカル環境で実行
- APIに送信するデータ量を制限

### 分析結果の解釈

## 🆘 トラブルシューティング

### よくある問題

**1. API Key エラー**
```
解決方法: .envファイルのAPIキーを確認
```

**2. パッケージインストールエラー**
```
解決方法: pip install --upgrade pip
```

**3. 日本語文字化け**
```
解決方法: CSVファイルをUTF-8で保存
```

**4. メモリ不足**
```
解決方法: データサイズを小さくするか、処理を分割
```
